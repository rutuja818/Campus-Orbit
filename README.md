# Campus Orbit

## Academic Calendar Management System

Campus Orbit is a web-based Academic Calendar Management System designed for colleges to manage academic events, approvals, notifications, and reports in a centralized way.

## Key Features

- Role-based system: Student, Staff, HOD, Principal, Admin
- Staff can create and manage academic events
- HOD and Principal approval workflow
- Event conflict detection
- Students can view approved events department-wise
- Automated email notifications and alerts
- PDF report generation
- Admin analytics dashboard
- Centralized academic event management

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Go / Golang
- Gin Framework
- JWT Authentication
